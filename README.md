# HTML & CSS 학습 요약 
- **(2025.02.03 ~ 2025.02.07)**

## HTML5
- **문서 구조 태그**: `<html>`, `<head>`, `<body>`
- **텍스트/링크/이미지**: `<h1>`~`<h6>`, `<p>`, `<a>`, `<img>`
- **목록과 표**: `<ul>`, `<ol>`, `<table>`
- **폼 요소**: `<input>`, `<button>`, `<form>`

## CSS3
- **텍스트/배경 스타일**: `color`, `background-color`
- **여백과 정렬**: `margin`, `padding`, `text-align`
- **레이아웃**: `display`, `flexbox`
- **반응형**: `@media query`
